## 전체 플로우 (진입 경로 2가지)

```
① 하단 네비 ✏️ → 전시 선택(04-01) ┐
② 전시 상세 "기록하기" ────────────┴→ 기록 작성 폼(04-05~07)
                                       ├ 감정 키워드 시트: GET /emotion-keywords
                                       ├ 미디어 추가: POST /files (RECORD_MEDIA)
                                       └ 작성 방식 선택
                                          ├ 직접 작성(04-08) → POST /records (DIRECT)
                                          └ AI 작성(04-09) → questions → compose → POST /records (AI)

전시가 목록에 없으면 → "전시 직접 추가하기"(04-02~04)
   포스터: POST /files (EXHIBITION_POSTER)
   전시관: GET /venues
   등록:   POST /exhibitions/custom → exhibitionId 로 위 기록 플로우 계속
```
