## 화면 → 도메인 매핑

| 와이어프레임 화면 | 담당 도메인 | 관련 폴더 |
|---|---|---|
| [01] 홈 (배너·곧 종료·신규·무료 섹션·종 아이콘) | exhibition, notification | 전시, 알림 |
| [02] 전시 탐색 (검색·지역/장르 필터·정렬) | exhibition | 전시 |
| [03] 전시 상세 (포스터·기간·장소·운영시간·관람료·지도·관심) | exhibition, bookmark | 전시, 북마크 |
| [04] 전시 관람 기록 (전시 선택·직접 추가·감정/미디어·직접/AI 작성) | record, exhibition, support | 기록, 전시, 공통 |
| [05] 아카이브 (그리드·정렬·상세) | record | 기록 |
| [06] 프로필 (보관함·감정 키워드·수정·설정) | user, bookmark, record | 유저, 북마크, 기록 |
| [07] 리마인드 (오늘의 여운·감정 다시 남기기·변화 요약) | remind, notification | 리마인드, 알림 |

## 도메인별 전체 API 목록 (베타/와이어프레임 기준)

> 상태: ✅ 기존 유지 · 🔧 기존 확장 · 🆕 신규 &nbsp;|&nbsp; 인증: 🔒 필수 · 🔓 선택(로그인 시 개인화) · ⚪ 불필요
> 전체 **32개** (✅ 11 · 🔧 7 · 🆕 14). 각 API 상세(요청/응답/에러)는 도메인 문서 링크 참조.

### 인증 auth — [문서](인증/README.md)

| 기능 | Method | Path | 인증 | 상태 |
|---|---|---|---|---|
| 소셜 로그인 | POST | `/api/v1/auth/login/{provider}` | ⚪ | ✅ |
| 게스트 로그인 | POST | `/api/v1/auth/guest` | ⚪ | ✅ |
| 토큰 재발급 | POST | `/api/v1/auth/refresh` | ⚪(refresh 쿠키) | ✅ |
| 로그아웃 | POST | `/api/v1/auth/logout` | 🔒 | ✅ |

### 유저 user — [문서](유저/README.md)

| 기능 | Method | Path | 인증 | 상태 |
|---|---|---|---|---|
| 내 프로필 조회 | GET | `/api/v1/users/me` | 🔒 | ✅ |
| 프로필 수정 | PUT | `/api/v1/users/me/profile` | 🔒 | ✅ |
| 알림 설정 조회 | GET | `/api/v1/users/me/notification-settings` | 🔒 | 🆕 |
| 알림 설정 수정 | PUT | `/api/v1/users/me/notification-settings` | 🔒 | 🆕 |
| 회원 탈퇴 | DELETE | `/api/v1/users/me` | 🔒 | 🆕 |
