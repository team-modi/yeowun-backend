# AI 임시저장(draft) — 캐싱·복원

> 이슈② — '질문으로 작성' 진행 상태(질문+답변+초안)를 Redis에 캐싱해 두고, 뒤로가기 후 재진입 시 복원한다.

**다이어그램이 필요한 이유**
- 상태 없는 AI 플로우 보완: `/questions`는 매 호출 질문을 **재생성(비결정적)**하므로, 캐싱 없이는 뒤로가기 후 같은 질문·답변을 되살릴 수 없다(→ 답변 유실을 "되돌아간다"로 체감).
- 캐시 포트/어댑터: 애플리케이션은 `AiDraftStore` 포트만 의존하고 Redis는 어댑터 세부(DIP). **Redis 장애 시 degrade**(save=no-op·find=empty) — 질문/감상문 생성·저장은 정상.
- 키·TTL: `ai:draft:{userId}:{exhibitionId}`, TTL 1h — 본인 draft만 접근, 방치 draft는 자동 만료.
- 프론트 연동: 뒤로가기 후 재진입 시 `GET /draft`로 상태를 채운다(UX 완성은 프론트 몫 — 별도 티켓).

```mermaid
sequenceDiagram
    actor 사용자
    participant RecordAiV1Controller
    participant RecordAiFacade
    participant AiDraftStore
    participant Redis

    Note right of 사용자: 인증된 사용자 (본인 draft만)

    rect rgb(245,237,224)
    Note over 사용자,Redis: 진행 중 자동 캐싱 (질문 생성·답변·초안 시점마다 덮어씀)
    사용자->>RecordAiV1Controller: POST /questions (질문 생성)
    RecordAiV1Controller->>RecordAiFacade: questions()
    RecordAiFacade->>AiDraftStore: save(질문)
    AiDraftStore->>Redis: SET ai:draft:{u}:{ex} (TTL 1h)
    사용자->>RecordAiV1Controller: PUT /draft (답변 자동저장)
    RecordAiV1Controller->>RecordAiFacade: saveDraft(질문+답변)
    RecordAiFacade->>AiDraftStore: save
    AiDraftStore->>Redis: SET
    사용자->>RecordAiV1Controller: POST /compose (초안 생성)
    RecordAiV1Controller->>RecordAiFacade: compose()
    RecordAiFacade->>AiDraftStore: save(질문+답변+초안)
    AiDraftStore->>Redis: SET
    end

    rect rgb(232,220,200)
    Note over 사용자,Redis: 뒤로가기 후 재진입 → 복원
    사용자->>RecordAiV1Controller: GET /draft?exhibitionId=
    RecordAiV1Controller->>RecordAiFacade: getDraft()
    RecordAiFacade->>AiDraftStore: find(u, ex)
    AiDraftStore->>Redis: GET
    alt 있음(만료 전)
        Redis-->>AiDraftStore: draft JSON
        AiDraftStore-->>RecordAiFacade: AiDraft
        RecordAiFacade-->>사용자: exists=true (질문·답변·초안)
    else 없음/만료/Redis 장애
        AiDraftStore-->>RecordAiFacade: empty
        RecordAiFacade-->>사용자: exists=false
    end
    end

    opt 감상문 저장 완료·작성 포기
        사용자->>RecordAiV1Controller: DELETE /draft?exhibitionId=
        RecordAiV1Controller->>RecordAiFacade: deleteDraft()
        RecordAiFacade->>AiDraftStore: delete
        AiDraftStore->>Redis: DEL
    end
```
