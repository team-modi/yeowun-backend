# venue 도메인 기능 목록

> 전시관(venue) 마스터. 개인 전시 직접 등록 시 전시관명 자동완성과 venueId 선택에 쓰인다.

| # | 기능명 | HTTP | 인증 | 설명 | 다이어그램 |
|---|---|---|---|---|---|
| 01 | 전시관명 자동완성 검색 | `GET /api/v1/venues?keyword=` | 필수 | 전시관명 부분 일치(대소문자 무시) 상위 20개 고정. keyword 공백/미입력이면 빈 목록(에러 아님) | [01-전시관명-자동완성.md](01-전시관명-자동완성.md) |

**비고**

- venue 도메인의 공개 API는 자동완성 검색 하나뿐이다. 전시관 등록/수정/삭제 API는 존재하지 않는다(마스터 데이터).
- `VenueErrorCode.VENUE_NOT_FOUND`(404)는 venue API에서는 발생하지 않는다. 개인 전시 등록(`ExhibitionFacade.registerCustom`)이 venueId로 전시관을 조회해 장소명·지역을 파생할 때, 없는 venueId면 이 에러를 던진다 — 해당 흐름의 다이어그램은 exhibition 도메인 소관.
