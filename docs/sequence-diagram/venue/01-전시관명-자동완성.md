# 전시관명 자동완성 검색

> 시나리오 2.4 — 사용자가 전시관명을 입력하면 자동완성 목록(상위 20개)이 뜬다.

**다이어그램이 필요한 이유**
- 조건 분기: keyword 공백/미입력이면 질의 없이 빈 목록을 반환한다(에러 아님)
- 규칙 3.4: 상한은 상위 20개 고정(커서 미적용), 부분 일치는 대소문자 무시 + soft-delete 행 제외

```mermaid
sequenceDiagram
    actor 사용자
    participant VenueV1Controller
    participant VenueFacade
    participant VenueRepository

    Note right of 사용자: 인증된 사용자

    사용자->>VenueV1Controller: 전시관명 검색 요청 (keyword)
    VenueV1Controller->>VenueFacade: 전시관명 자동완성 (상한 20)

    VenueFacade->>VenueRepository: 이름 부분 일치 검색
    alt keyword 공백/미입력
        VenueRepository-->>VenueFacade: 빈 목록 (질의 생략)
    else keyword 존재
        VenueRepository-->>VenueFacade: 살아있는 전시관 상위 20개
    end

    VenueFacade-->>VenueV1Controller: 결과 반환
    VenueV1Controller-->>사용자: 성공 (결과 없으면 빈 배열)
```
